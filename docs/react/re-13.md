# createElement 与 cloneElement 的区别是什么？

?> 英：What is the difference between createElement and cloneElement?

**答：**

> `createElement` 函数是 `JSX` 编译之后使用的创建 `React Element` 的函数，而 `cloneElement` 则是用于复制某个元素并传入新的 `Props`。
