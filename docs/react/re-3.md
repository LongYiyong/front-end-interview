# 什么情况下你会优先选择使用 Class Component 而不是 Functional Component？

?> 英：When would you use a Class Component over a Functional Component?

**答：**

> 在组件需要包含内部状态或者使用到生命周期函数的时候使用 `Class Component` ，否则使用函数式组件。
